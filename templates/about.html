About Us

SPYGUARD is a Network Intrusion Detection System designed to identify and classify network attacks using advanced machine learning techniques. The system employs the Synthetic Minority Over-sampling Technique (SMOTE) to address class imbalance in the training data, improving detection accuracy. It uses a Random Forest (RF) algorithm as the core classifier to detect various types of network intrusions effectively.

The system is trained and evaluated on the NSL-KDD dataset, a widely recognized benchmark dataset for network intrusion detection research. SPYGUARD provides users with an intuitive interface to upload network traffic data, analyze it, and view detailed detection results.
